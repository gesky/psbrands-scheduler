Política de Privacidade

Última atualização: maio de 2026

Este aplicativo (Scheduler) é uma ferramenta interna desenvolvida pela OJO Studio para agendamento de publicações em redes sociais para seus clientes. O uso é restrito a operadores autorizados da OJO Studio.

Dados coletados

O aplicativo acessa, via Meta Graph API e LinkedIn API, informações das páginas e contas de redes sociais dos clientes autorizados, exclusivamente para fins de publicação e agendamento de conteúdo. Nenhum dado pessoal de terceiros é coletado ou armazenado.

Uso dos dados

  • Autenticação com o Facebook/Instagram via OAuth
  • Publicação e agendamento de posts em páginas autorizadas
  • Os tokens de acesso são armazenados localmente no navegador do operador (localStorage) e nunca transmitidos a servidores de terceiros

Compartilhamento de dados

Nenhum dado é compartilhado com terceiros. O aplicativo comunica-se exclusivamente com a API oficial do Meta e da LinkedIn.

Retenção de dados

Os tokens de acesso ficam armazenados localmente no navegador e podem ser removidos a qualquer momento limpando os dados do site.

Contato

Para dúvidas sobre esta política, entre em contato com a OJO Studio.
